# Products

Shippable customer-facing code lives under **`products/`**, grouped by **business org** (`org-shopping`, `org-growth`, `org-media`, `org-social`, `org-platform`, `org-lifestyle`) and **vertical** (for example `ads` under growth, `ai` under platform).

Place new experiments in `org-platform/local/` (see `./scripts/new-app.sh`) or add a new org/vertical when a product line warrants its own subtree.
