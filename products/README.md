# Products

**At 216Labs we are building the toolkit for production grade vibes.** Shippable customer-facing code lives under **`products/`**, grouped by **business org** (`org-shopping`, `org-growth`, `org-media`, `org-social`, `org-platform`, `org-lifestyle`) and **vertical** (for example `ads` under growth, `ai` under platform).

Place new experiments in `org-platform/local/` (see `./scripts/new-app.sh`) or add a new org/vertical when a product line warrants its own subtree. Minimal **toolkit demos** (hello-world apps) live in **`org-platform/toolkit-demos/`** — see `docs/TOOLKIT.md`.
