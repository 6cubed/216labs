# Toolkit demos

Minimal **hello-world** apps that prove the monorepo pipeline (manifest → Docker → Caddy → admin) end to end.

- **`hello-nextjs/`** — Next.js (App Router, standalone output)
- **`hello-flask/`** — Flask + Gunicorn

See **`docs/TOOLKIT.md`** for how this relates to a client-agnostic starter fork and how to run a small stack locally.
